# 効果音(SE) 置き場

ここに **SEファイル** を置くと、対応するタイミングで1発鳴ります（未配置なら無音でスキップ）。

ファイル名は `config/config.story.yaml` の `audio.se.*` に合わせる（既定）:

| トリガー | 既定ファイル名 | 鳴るタイミング |
|---|---|---|
| `intro` | `intro.mp3` | 動画の冒頭 |
| `outro` | `outro.mp3` | outro章（締め）の頭 |
| `flash` | `flash.mp3` | 各ネタの切替（最初の発言）|
| `surprise` | `surprise.mp3` | ずんだもん(聞き手)の驚き「ええー！？」の頭だけ |

- 形式：`.mp3` / `.wav` / `.m4a` / `.ogg`
- 全体音量は config の `audio.se_volume`、連発抑制は `audio.se_min_gap`（直前SEからこの秒数以内は鳴らさない）
- 不要なトリガーは config の該当行を消すかファイルを置かなければOK（自動でスキップ）

## 無料・商用可・収益化可の音源（例）
- **効果音ラボ** … 定番。商用・収益化OK・帰属不要
- **魔王魂** … 商用OK

※ 従量課金・有料素材は使わない。帰属が要る音源だけ `docs/story/credits.txt` に追記すること。
