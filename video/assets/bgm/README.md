# BGM 置き場

ここに **BGMファイル** を置くと動画全体に薄くループで流れます（未配置なら無音でスキップ）。

- ファイル名：`config/config.story.yaml` の `audio.bgm.file`（既定 `bgm.mp3`）に合わせる
- 形式：`.mp3` / `.wav` / `.m4a` / `.ogg`
- 音量・フェードは config の `audio.bgm.volume` / `audio.bgm.fade` で調整（render を見て詰める）

## 無料・商用可・収益化可の音源（例）
- **DOVA-SYNDROME** … 無料BGM。規約に沿えば帰属任意
- **魔王魂** … 無料BGM/SE。商用OK

※ 従量課金・有料素材は使わない。帰属が要る音源だけ `docs/story/credits.txt`（概要欄用）に追記すること。
