使用中の構図は削除できません。slotの「構図」で割り当ててから台本のslot framingに使います。
center(0.500, 0.500) / width 1.000
コピー元の width と cy を、現在編集中の枠へ適用します。左右位置(cx)は変えません。
default / leftFocus / rightFocus の3枠をシーン共通の構図として保存します。話者フォーカスONのターンでは、話者の立ち位置に応じて枠が切り替わります。
