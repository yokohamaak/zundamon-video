# 第1話「AIが大丈夫って言ったのだ。」

ストーリー動画エディタ用 台本（ドラフト）。
口調: ずんだもん「〜のだ」／めたん「〜よ・わ・ね」。表情は normal / happy / surprise / trouble / panic の5種。
営業・部長はモブ（1枚絵・後付け、声はVOICEVOX別キャラ割当）。

## シーン一覧（背景）

| key | label | 内容 | layered |
|---|---|---|---|
| `office` | オフィス | 平常の事務所。手前にデスク/モニタ枠を置きキャラ下半身を隠す | ○（front有り） |
| `home` | 自宅 | 帰宅カット | － |
| `server_room` | サーバールーム | 薄暗くラックが並ぶ。独自設定＝インフラの象徴 | 台本確定後に判断 |
| `rooftop` | 屋上・夕方 | 空が広い夕景。締めの余韻 | － |

**演出で出す（背景は作らない）**: 障害中の緊迫＝赤フラッシュ＋アラートテロップ／前日パート＝セピア調の回想トーン。
