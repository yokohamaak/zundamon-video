# ストーリーエディタ MVP 動作確認チェックリスト

起動: Mac で `./run-story story` → ブラウザで `http://localhost:8771`
（ヘッドレスで実装したため実ブラウザ動作は未検証。引っかかった項目を報告 → 修正）

## 1. 起動・初期表示
- [ ] ページが開く／エラーで落ちない
- [ ] 左に台本一覧（第1話の52ターン）が出る
- [ ] 一覧が**場面ごとにグループ**（▸ office など）で表示される
- [ ] 各行に**話者アイコン**（ずんだ/めたん/営業/部長の顔）が出る／壊れ画像でない
- [ ] 演出バッジ（emphasis/shake/flashback/insert）が該当ターンに付く

## 2. 左ペイン（一覧操作）
- [ ] 行クリックで選択 → 右の詳細に内容が反映される
- [ ] **＋追加**：選択行の後ろに空ターンが入る
- [ ] **削除**：選択ターンが消える
- [ ] **↑ / ↓**：並べ替えできる

## 3. 右ペイン「基本」タブ
- [ ] 話者（ドロップダウン）変更 → 左一覧のアイコン/名前が変わる
- [ ] セリフ（テキスト）編集 → 左一覧のセリフ先頭が変わる
- [ ] シーン（ドロップダウン＋自由入力）変更できる
- [ ] 表情（ドロップダウン）変更できる
- [ ] 演出トグル（emphasis / shake / flashback）ON/OFFできる → バッジ反映
- [ ] telop（テキスト）・pause（数値）入力できる
- [ ] enter / exit（マルチ選択）＋ exitDir（left/right）設定できる

## 4. 右ペイン「インサート」タブ
- [ ] kind 切替（なし/warning/ok/chat/teamchat）で入力欄が切り替わる
- [ ] warning：title / text 入力
- [ ] chat：user／ai（**行の追加・削除**）／highlight
- [ ] teamchat：channel／messages（**from選択・text・highlight、行の追加・削除**）
- [ ] 既存ターン（コールドオープンの teamchat）が**正しく読み込まれて表示**される

## 5. 保存・往復
- [ ] **💾保存**を押す → 保存ステータスが出る／エラーにならない
- [ ] ブラウザを**リロード**して、編集内容が**残っている**（story-01.json に書けている）
- [ ] 保存後 `./run-story render` で動画に反映される（最終確認）

## 6. その他
- [ ] 日本語が文字化けしない（一覧・詳細・保存ファイル）
- [ ] 音声生成／書き出しボタンは**押せない（disabled）**でOK（次フェーズ）
